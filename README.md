# 🏢 Employee Leave Management System

A simple and user-friendly **Employee Leave Management System** developed using **Python, Streamlit, SQLite, and Pandas**. This application allows employees to apply for leave online while enabling administrators to manage, approve, or reject leave requests through an interactive dashboard.

---

## 📌 Features

### 👨‍💼 Employee Module

* Apply for leave
* Select department
* Choose leave type
* Select start and end dates
* Provide leave reason
* Submit leave request

### 🛠️ Admin Module

* View all leave requests
* Dashboard statistics
* Approve leave requests
* Reject leave requests
* Automatic status updates

## 💾 Database

The project uses **SQLite** as the database.

The database stores:

* Employee Name
* Department
* Leave Type
* Start Date
* End Date
* Reason
* Leave Status
